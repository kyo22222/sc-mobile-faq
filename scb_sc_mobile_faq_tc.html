設定6位數字密碼、離線密碼

SC Mobile Key（流動保安編碼）是甚麼？

SC Mobile Key是一個在渣打流動理財App的增強功能使您能夠驗證網上理財服務的登入和交易。 在您的手機上綁定及啟用SC Mobile Key後，不再需要使用一次有效密碼或/及交易密碼。

使用SC Mobile Key有甚麼好處？

透明：不再需要輸入一次有效密碼來驗證您的銀行交易或網上股票投資服務。 讓SC Mobile Key服務為您處理。

安全：執行高風險交易如新增收款人時，SC Mobile Key服務會以數碼方式驗證您的高風險交易增加一層保障。

任何地方：擔心您在海外而不能使用SC Mobile Key服務？ 我們的SC Mobile Key服務即使在沒有數據連接的情況下仍可以使用。 在離線情況下，您可以在使用問答回應碼下取得離線交易密碼來執行高風險的交易驗證。

如何啟用SC Mobile Key？

請先下載最新的SC Mobile App，以您的網上理財用戶名稱及密碼或指紋認證（Touch ID）/ 生物認證（Face ID）登入後，其後頁面便會提示您如何綁定及啟用SC Mobile Key服務。

為甚麼我需要設定一個SC Mobile Key 6位數字密碼？

渣打SC Mobile Key 6位數字密碼為您提供額外的保障來執行高風險交易驗證（例如新增收款人、轉賬額設定及更新個人資料等等）或取得離線密碼。 只有您知道該6位數字密碼來執行這些類型的銀行交易。

當我更換手機時是否需要重新設定SC Mobile Key？

需要。SC Mobile Key只能綁定一部手機，如您想將SC Mobile Key轉換至新手機，請重新設定SC Mobile Key。

您將以短訊或電郵方式收取訊息。

您將以短訊或電郵方式收取訊息。

我會收到甚麽類型的推送訊息？

訊息種類包括服務訊息、交易提示、優惠及產品資訊。如銀行戶口的繳付賬單、提款、轉賬、購物、退票及信用卡的還款日提示。

我可否經SC Mobile App申請綜合存款戶口？

您必須符合以下所有條件方能透過SC Mobile App申請綜合存款戶口：

1. 未持有以下任何渣打銀行（香港）有限公司之存款戶口︰

  • 往來／支票戶口
  • 綜合存款戶口
  • 定期存款戶口
  • 投資戶口

2. 持有有效香港身份證；

3. 香港永久性或非永久性居民，並年滿十八歲或以上；

4. 非美國居民、非美國公民及非持有美國永久居民身份證（綠卡）；

5. 開立此戶口作個人用途

實際所需時間或會因個別情況而有所不同。

我持有渣打信用卡，我可否經SC Mobile App申請綜合存款戶口？

為甚麼要拍攝身份證的正面及背面?為甚麼總是不成功？

拍攝身份證的正面及背面有助我們驗證您的香港身份證，讓您無需親自前往 我們的分行進行身份驗證。拍攝時請確保有足夠及穏定的光源，沒有被反光面或陰影遮掩，並確認身份證的影像清晰可見、不失焦。

有些因素有機會導致身份驗證不成功，例如：強光、身份證上的劃痕、 損耗等，都可能影響系統對您的身份證及自拍照進行驗證及對比之準確性。 如以上因素令系統未能檢測到有效的香港身份證，我們未能透過系統完成身份核實程序，請親臨我們任何一家分行辦理開戶手續。

我們可能需要您提供一些證明文件或回答一些跟進問題。請你於30天內携同您的身份證明文件親臨我們任何一間分行以完成開戶手續。

為節省等候時間，您可在我們網站預約到指定分行開戶。

成功開立綜合存款戶口後，我們會為你發出一張渣打銀聯單幣提款卡。請您於收到提款卡後，立即登入網上理財確認收妥新卡。

請留意SC Mobile App的戶口設定開放時間為星期一至日早上8時至晚上10時。
