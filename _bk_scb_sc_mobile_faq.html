由現行SC Mobile App （版本6.31.5或以下） 升級至新版 SC Mobile App （版本7.2.0或以上）

SC Mobile App 升級時會遇到的問題

升級版SC Mobile 應用程式有甚麽新功能?

我們聽到了客戶的聲音。升級版SC Mobile 應用程式 將會比以往更個人化及流暢，全面提升客戶的數碼銀行體驗，並為客戶帶來包括但不限於以下的新體驗:

  • 煥然一新的用戶界面
  • 管理轉賬及繳付賬單，並增加收款人
  • 探索最新、熱門、個人化的產品與服務
  • 一站式自助服務與通訊，讓您無需在不同界面找尋所需服務。
怎樣得知我正在使用的是現行SC Mobile 應用程式或升級版 SC Mobile 應用程式？

若版本編號為6.31.5 或以下，即代表您正在使用現行的SC Mobile 應用程式。若版本編號為7.2.0 或以上，即代表您正在使用升級版SC Mobile 應用程式。

如何查看SC Mobile 應用程式版本編號？

方法1：您可以前往流動裝置的應用程式商店，搜尋“SC Mobile Hong Kong”，選擇SC Mobile Hong Kong 以查看更多内容，並在“應用程式資料“中找到版本編號。

方法2：打開現行SC Mobile 應用程式，點選畫面左上角的菜單符號，點擊「關於我們」，畫面下方將出現版本編號。

方法3：打開升級版SC Mobile 應用程式，點選畫面右上角問號按鈕，畫面下方將出現版本編號。

我是現行的SC Mobile 應用程式用戶。我是否需要重新登記SC Mobile賬戶或重新設定SC Mobile Key（流動保安編碼）/ 指紋認證 （Touch ID）/ 生物認證（Face ID）以使用升級版 SC Mobile 應用程式？

不需要。您只需於流動裝置的應用程式商店更新SC Mobile 應用程式即可使用升級版SC Mobile 應用程式。所有用戶名稱、密碼、SC Mobile Key、指紋認證 、生物認證、推送訊息的設定將維持不變。您無需重新登記SC Mobile 賬戶或重新設定SC Mobile Key。

我甚麽時候可以開始使用升級版SC Mobile 應用程式？

密切期待！由於更新需時，您可於11月中後前往流動裝置應用程式商店或於我們官網下載升級版SC Mobile 應用程式 。特選客戶或於11月初即可預先體驗我們升級版 SC Mobile。

現行之SC Mobile 應用程式 將於甚麽時候停止使用？

若您未有自行升級至升級版 SC Mobile 應用程式，我們將於2023年12月8日起將現有用戶升級至升級版SC Mobile 應用程式，詳情請留意網站的公佈。

下載升級版SC Mobile 應用程式後，我能否再使用現行的SC Mobile 應用程式？

很抱歉，為統一流程及配合銀行系統進行全面提升，客戶下載升級版SC Mobile 應用程式後將無法再使用現行的SC Mobile 應用程式版本。

是次升級會否對網上理財造成影響？

沒有。

爲何升級版SC Mobile 應用程式沒有於登入畫面顯示快捷鍵？

為配合界面及登入速度的提升，我們已升級用戶界面。

  • 轉賬或提款（QR Cash）功能可於「轉賬及付款」主頁或頁面查找
  • 投資或保險可於「投資」頁面查找
  • 網上櫃位取票服務請按登入頁面上右上角按鈕或登入後右上角之按鈕選項內查找。

我的流動裝置與升級版SC Mobile 應用程式兼容嗎？爲何升級後，我未能使用SC Mobile 應用程式？

SC Mobile 應用程式支援的流動裝置包括：

  • iPhone、iPad及iPod Touch（使用 iOS／iPadOS系統版本 14.0 或以上，不包括Beta版本）
  • Android™ 手機（使用Android系統版本 9.0 或以上，不包括Beta版本）
  • Touch ID／Face ID認證功能只適用於支援Touch ID／Face ID的Apple iPhone、iPad及iPod Touch型號
  • 指紋認證只適用於兼容Android系統版本 9.0 或以上的Android™設備。

若您的流動裝置符合上述要求但未能成功使用SC Mobile，請提升您的系統版本以體驗全新的 SC Mobile。
為配合系統進行全面提升，我們恕未能為未達到上述兼容性要求的流動裝置進行升級。 若您的流動裝置未能符合上述要求，歡迎您使用渣打網上理財以繼續使用我們服務。

我正在使用Android™ 手機，爲何打開升級版SC Mobile 應用程式時會出現系統錯誤？

因系統安全性原因，部分使用Android™ 手機而未有使用官方鍵盤的用戶將暫時未能使用升級版SC Mobile 應用程式。請先切換至官方鍵盤或使用渣打網上理財。

我沒有自動櫃員機私人密碼/ 電話理財私人密碼，我應該如何登記SC Mobile 應用程式？

如沒有自動櫃員機私人密碼/ 電話理財私人密碼，您仍然可以根據信用卡客戶步驟登記網上理財，但您只能享用有關信用卡理財服務以查閱信用卡結餘及交易/ 查閱電子月結單/ 兌換「 360°全面賞」積分/ 現金回贈的服務。
如您想享用全面的數碼銀行服務，請聯絡客戶服務熱線以便安排郵寄自動櫃員機私人密碼/ 電話理財私人密碼，或親臨分行領取。

我可否於海外使用 SC Mobile 應用程式？

無論您身在全球任何地方，均可輕鬆便利地享受我們的流動理財，惟部分受國際經濟制裁的國家除外。我們矢志遵從有關制裁。因此，我們概不允許在古巴、伊朗、朝鮮、蘇丹、敍利亞、克里米亞和塞瓦斯托波爾直接或間接使用我們的產品及服務。
在您前往古巴、伊朗、朝鮮、蘇丹、敍利亞、克里米亞或塞瓦斯托波爾期間，將不可使用我們的產品與服務。待您離開這些國家之後，即可繼續使用。

很抱歉。由於已被破解的裝置會令客戶較容易蒙受欺詐交易的風險，為保障客戶利益，當我們的系統偵測到您的流動裝置已被破解，您將不能使用SC Mobile 應用程式。為確保你的賬戶及個人資料安全得到保障，請使用其他未被破解的裝置登入SC Mobile 應用程式。

我如何將SC Mobile Key（流動保安編碼）轉移到新裝置？

請於新裝置登入應用程式，並輸入個人網上理財的用戶名稱及密碼，選擇切換流動保安編碼至新裝置。

登記SC Mobile Key的步驟如下：
步驟1：登入您的SC Mobile 應用程式後，您將會收到提示登記SC Mobile Key。
步驟2：輸入以電郵形式發送給您的一次有效密碼。
步驟3：輸入以短訊形式發送給您的一次有效密碼。
步驟4：設定您的6位數字密碼。
步驟5：成功登記。您現可透過您的6位數字密碼，在新裝置驗證所有交易。
當您成功於新裝置啟動流動保安編碼，舊裝置的流動保安編碼及其6位數字密碼將會失效。

SC Mobile支援的流動裝置包括：
步驟 1：輕觸流動理財登入版面的面孔/ 手指圖示，選擇「立即啟動」。

步驟3：於「啟動指紋認證」或「啟動生物認證」的左邊切換按鈕向左拉動以停止使用指紋認證（Touch ID）/ 生物認證（Face ID）。

SC Mobile Key（流動保安編碼）是甚麼？

SC Mobile Key是一個在渣打流動理財應用程式的增強功能使您能夠驗證網上理財服務的登入和交易。 在您的手機上綁定及啟用SC Mobile Key後，不再需要使用一次有效密碼或/及交易密碼。

使用SC Mobile Key有甚麼好處？

透明：不再需要輸入一次有效密碼來驗證您的銀行交易或網上股票投資服務。 讓SC Mobile Key服務為您處理。
安全：執行高風險交易如新增收款人時，SC Mobile Key服務會以數碼方式驗證您的高風險交易增加一層保障。
任何地方：擔心您在海外而不能使用SC Mobile Key服務？ 我們的SC Mobile Key服務即使在沒有數據連接的情況下仍可以使用。 在離線情況下，您可以在使用問答回應碼下取得離線交易密碼來執行高風險的交易驗證。

如何啟用SC Mobile Key？

請先下載最新的SC Mobile 應用程式，以您的網上理財用戶名稱及密碼或指紋認證（Touch ID）/ 生物認證（Face ID）登入後，其後頁面便會提示您如何綁定及啟用SC Mobile Key服務。

為甚麼我需要設定一個SC Mobile Key 6位數字密碼？

渣打SC Mobile Key 6位數字密碼為您提供額外的保障來執行高風險交易驗證（例如新增收款人、轉賬額設定及更新個人資料等等）或取得離線密碼。 只有您知道該6位數字密碼來執行這些類型的銀行交易。

當我更換手機時是否需要重新設定SC Mobile Key？

需要。SC Mobile Key只能綁定一部手機，如您想將SC Mobile Key轉換至新手機，請重新設定SC Mobile Key。

推送訊息的作用是甚麽?

我在哪裡可以設定我的每日轉賬限額？為甚麼我的每日轉賬限額被預設/ 重設為港幣0元？

您可透過渣打SC Mobile 應用程式「轉賬及付款」內的「每日轉賬額設定」更改每日網上轉賬限額。
基於網上保安理由及保障您的銀行戶口，我們會為全新登記網上及流動理財的客戶預設每日網上轉賬額為港幣0元（軟件更新除外）。此外，若您一段時間並未透過網上理財或SC Mobile 應用程式使用其中一個轉賬類別限額（例如「轉賬至第三者或其他本地銀行戶口」），該轉賬類別限額亦會被重設為港幣0元。

使用信用卡作轉賬會收取任何費用嗎?

若閣下在於該曆月（以交易日計算為準）憑渣打/ MANHATTAN信用卡透過不時生效 之電子銀行服務、個人對個人(P2P)支付服務或流動裝置/應用程式/電子轉賬平台作任 何轉賬/增值到指定賬戶（包括但不限於八達通銀包）累積轉賬多於25,000港元，本行會就25,000港元其後之所有轉賬金額收取3.5%手續費並誌賬於其中一張累積最多 轉賬金額之有效信用卡賬戶內。如當月最後一天為星期日，任何當天之有關交易將計 算於下一個月份內。如持卡人於曆月的第一天以及在系統更新完成之前或之間進行交易，則該交易將計算於前一個曆月進行。持卡人於進行電子轉賬交易而累積轉賬金額多於最多轉賬金額時，網上銀行及/或流動應用程式將顯示可用而免收手續費之轉賬金額及其後因而衍生之手續費。持卡人於 確認進行轉賬時，則被視為承認並同意網上銀行及/或流動應用程式顯示而本行按本 條款或其他現行促銷條款及細則不時指定的手續費（如適用）。
閣下亦可參閱最新超額電子轉賬手續費推廣計劃條款及細則。

透過SC Mobile 應用程式新增本地賬戶/賬單收款人與網上理財會否有分別？

兩者並無分別，而且SC Mobile 應用程式及網上理財均會顯示同一收款人列表。

如何於 SC Mobile 應用程式 登記本地賬戶/賬單?

你可以透過SC Mobile 應用程式「轉賬及付款」內的「我的收款人」, 選擇「新增收款人」以作登記。

「本地轉賬」與「SC Pay （轉數快）」有甚麼分別？

「本地轉賬」以銀行戶口號碼作轉賬，並且須事先透過SC Mobile 應用程式或網上理財登記該收款人，其每日轉賬限額屬第三者戶口轉賬之港幣1,000,000元。
「SC Pay （轉數快）」以手提電話號碼、電郵地址或轉數快識別碼作轉賬，無需作任何收款人登記，其每日轉賬限額屬非登記受款人轉賬之港幣300,000元。

「本地轉賬」提供何種類型的轉賬服務?

「本地轉賬」提供轉賬至已登記的第三者渣打戶口或其他本地銀行戶口。此項服務適用於港幣及人民幣轉賬。

為甚麼我未能於選擇支賬戶口作轉賬時看見我的信用卡?

請於選擇收款人頁面查看您的收款人是否經由簽署確認登記之戶口（以*表示），信用卡目前並不支援轉賬至經由簽署確認登記之戶口。

為甚麼我的轉賬會因「收款人名稱不正確」而被拒？

收款機構處理轉賬指示時可能會就其紀錄核對收款人名稱及戶口號碼等資料。為免轉賬被拒，請與收款人確認其於收款機構的戶口全名。如需更新，請重新於SC Mobile 應用程式或網上理財新增收款人。

為甚麼我的收款人會被刪除？

於 SC Mobile 應用程式或網上理財登記之收款人戶口，若於十二個月內沒有任何從銀行戶口成功轉賬之紀錄，該收款人將會被刪除。

誰可以登記 SC Pay「轉數快」?

成功登記 SC Pay「轉數快」，您需要：

  1. 成為渣打網上理財客戶， 並已登記使用SC Mobile 應用程式;
  2. 在銀行持有有效手機號碼或電郵地址; 和
  3. 持有合資格的個人或聯名（任何一方簽署）之港幣或人民幣戶口（支票賬戶，儲蓄賬戶或綜合存款賬戶）

我已經持有包含港幣及人民幣的綜合存款戶口，但為甚麼於SC Pay「轉數快」輸入轉賬金額時只顯示港幣/人民幣？

請確保該綜合存款戶口已綁定為SC Pay登記戶口，並且其港幣及人民幣戶口均已啟動。

我能否以港幣戶口作人民幣轉賬，或轉賬人民幣至收款人的港幣戶口？

SC Pay「轉數快」目前只支援從已登記SC Pay的戶口作相同貨幣轉賬。如收款人的「轉數快」戶口並非已包括港幣及人民幣的綜合戶口，轉賬例如人民幣至港幣戶口可能會被拒。

甚麼是「轉數快」預設收款銀行？我的付款人會否知道我的預設收款銀行？

除非您的「轉數快」付款人指定轉賬至某間您已登記的銀行，否則款項將會傳送至您的預設收款銀行，而且不會向付款人透露收款銀行名稱。

我能否於SC Mobile 應用程式查看其他銀行的「轉數快」登記？如何將渣打銀行更改為「轉數快」預設收款銀行？

您可於「管理」下的「SC Pay登記及設定」查閱以該手提電話號碼或電郵地址登記「轉數快」的其他銀行名稱並移除其紀錄。
如您尚未設定渣打銀行為「轉數快」預設收款銀行，您可於該頁面所顯示的手提電話號碼或電郵地址側旁點開選項，並選擇「設置渣打銀行為預設收款銀行」。

我可以使用信用卡作SC Pay登記並收取「轉數快」款項嗎？

您須以渣打儲蓄/ 支票戶口作SC Pay登記，信用卡只可用作港幣支賬而非收取款項。

我可以透過「繳付渣打信用卡」繳交親友的渣打信用卡嗎?

「轉賬紀錄」及「預設轉賬」的分別是甚麼？

「轉賬紀錄」列出已經執行之轉賬及付款，而「預設轉賬」所顯示的相反為尚待執行的指示。您可以於預定日期與收款人或商戶進行交易。此服務適用於我的戶口轉賬, 本地轉賬, 海外及跨境轉賬, 繳付賬單及信用卡。
您只需要簡單地選擇交易日期，我們便會在該日期自動為您進行轉賬。您必須確保在交易日之一個工作天前，您的賬戶内有足夠進行交易的款項或信用額（若您使用信用卡支付）。

「預設轉賬」暫時只會列出尚待執行之本地轉賬、海外及跨境轉賬、繳付賬單、繳付渣打信用卡，請登入網上理財以查看或更改常行指示。

請登入網上理財以更改預設轉賬或繳費的指示。

    • 澳洲 - 6位數字BSB Code

    • 美國 - 9位數字ABA Routing Number
    • 其他地區 - 11位數字 SWIFT Code

  3. 收款人姓名 （必須與收款銀行記錄中的姓名相同）
  4. 收款人地址 （包括省、城市和郵政編碼等）

我可否經SC Mobile 應用程式申請綜合存款戶口？

您必須符合以下所有條件方能透過SC Mobile 應用程式申請綜合存款戶口：

  1. 未持有以下任何渣打銀行（香港）有限公司之存款戶口︰

    • 往來／支票戶口

  3. 香港永久性或非永久性居民，並年滿十八歲或以上；
  4. 非美國居民、非美國公民及非持有美國永久居民身份證（綠卡）；
  5. 開立此戶口作個人用途
    6. 實際所需時間或會因個別情況而有所不同。

我持有渣打信用卡，我可否經SC Mobile 應用程式申請綜合存款戶口？

可以，只要您現時未持有任何渣打存款戶口及符合以上其他要求，您便可以使用SC Mobile 應用程式開立戶口。成功開戶後，您的渣打信用卡個人資料將根據此申請表格自動同時更新。

我曾經持有渣打戶口或信用卡，但我已將它們全部取消 。我現在可否使用此服務？

可以，只要您現時成為我們新客戶及符合以上其他要求，您即可以透過渣打流動理財應用程式進行開戶。

為甚麼要拍攝身份證的正面及背面?為甚麼總是不成功？

拍攝身份證的正面及背面有助我們驗證您的香港身份證，讓您無需親自前往 我們的分行進行身份驗證。拍攝時請確保有足夠及穏定的光源，沒有被反光面或陰影遮掩，並確認身份證的影像清晰可見、不失焦。有些因素有機會導致身份驗證不成功，例如：強光、身份證上的劃痕、 損耗等，都可能影響系統對您的身份證及自拍照進行驗證及對比之準確性。
如以上因素令系統未能檢測到有效的香港身份證，我們未能透過系統完成身份核實程序，請親臨我們任何一家分行辦理開戶手續。

請留意SC Mobile 應用程式的戶口設定開放時間為星期一至日早上8時至晚上10時。
如您未能完成戶口設定，您經SC Mobile 應用程式新開立之綜合存款戶口將維持一般儲蓄存款年利率及您的理財計劃會維持為Easy Banking。

甚麼是myRM？

myRM是一個電子銀行軟件使您能夠聯繫您的專屬客戶經理。

誰能使用myRM？

myRM是一個讓渣打香港「優先理財」客戶與專屬客戶經理溝通的官方聊天平台。

myRM的服務時間是甚麼？

服務時間與分行營業時間相同。另一方面，亦建議您可提前與您的專屬客戶經理安排預約所想的時間和日期。

myRM的服務範圍包括甚麼？

myRM使您可以在安全登錄後，通過文字及語音對話與我們聯繫。透過myRM，您的客戶經理可以與您分享產品文件，預約進行投資分析。
